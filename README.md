# Gator - RSS Feed Aggregator CLI

Gator is a multi-user CLI application written in Go that allows you to:

- Add RSS feeds from across the internet
- Follow and unfollow feeds
- Aggregate and store posts in a PostgreSQL database
- Browse posts directly in your terminal

---

## Requirements

Before installing Gator, make sure you have:

- Go (1.20+ recommended)
- PostgreSQL (v15+ recommended)

---

## Installation

Install the CLI using Go:

```bash
go install github.com/ShiriezH/Gator@latest
